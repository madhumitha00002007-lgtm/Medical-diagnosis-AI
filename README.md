# 🩺 AI Medical Diagnosis Expert System

An AI-based Medical Diagnosis Expert System developed using Python. This project analyzes user-selected symptoms and provides a preliminary diagnosis along with basic care recommendations through a simple rule-based approach.

## 🩺 Sample Diagnosis

The system allows users to:

- Select a symptom
- Analyze possible conditions
- Display a preliminary diagnosis
- Suggest basic care recommendations

Example:

- Symptom: Vomiting
- Suspected Condition: Food Poisoning
- Recommendation: Stay hydrated, drink ORS, and eat bland food.
